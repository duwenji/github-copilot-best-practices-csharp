# 06 リファレンス | Reference

## 📚 このセクションについて

**対象**: 全員  
**用途**: 辞書的・カタログ的な参照資料  
**学習時間**: 必要なときに読む（随時）

---

## 🎯 このセクションの役割

「**あ、あれどうやるんだっけ？**」というときに、すぐ調べられる参照資料集です：

- 🔧 **Troubleshooting** — よくあるエラーと解決策
- ⚡ **Quick Reference** — よく使うプロンプト・コマンド
- 📊 **Feature Matrix** — ロール別の推奨機能
- 📋 **Project Templates** — プロジェクト種別のベストプラクティス

---

## 📖 コンテンツ構成

### [01-troubleshooting.md](01-troubleshooting.md)
**用途**: エラーや困ったときの解決策  
**掲載内容**:
- Copilot が反応しない
- 不適切なコードが提案される
- セキュリティ上の懸念
- パフォーマンス問題
- IDE 固有の問題

**時間**: 問題が発生したときに参照（5-10 分）

---

### [02-quick-reference.md](02-quick-reference.md)
**用途**: よく使うプロンプト・ショートカット集  
**掲載内容**:
- キーボードショートカット一覧
- よく使うプロンプトテンプレート
- 改行・フォーマット例
- よくある質問 FAQ

**時間**: 毎日はじめに確認（1-2 分）

---

### [03-feature-by-role-matrix.md](03-feature-by-role-matrix.md)
**用途**: 自分のロールに最適な機能が何か一目瞭然  
**掲載内容**:
- ロール × 機能 マトリックス
- 各ロール最高の ROI 機能
- 習得優先度ランキング
- 使用シーン集

**時間**: キャリア計画時に確認（3-5 分）

---

### [04-project-templates.md](04-project-templates.md)
**用途**: プロジェクト種別ごとのベストプラクティス  
**掲載内容**:
- Web API プロジェクト
- Entity Framework Core プロジェクト
- Console アプリケーション
- Class Library
- テストプロジェクト

**時間**: 新規プロジェクト開始時に確認（5-10 分）

# 02 コア機能 | Core Features

## 📚 このセクションについて

**対象**: 初心者～中級者  
**学習時間**: 2-3 時間（機能ごと）  
**前提知識**: `01-foundations` の基本操作習得済み

---

## 🎯 このセクションのゴール

GitHub Copilot の**4つの主要機能**を順序立てて学び、各機能の使い分けができるようになります：

- ✅ **Inline Completions** — コード入力中の自動補完
- ✅ **Copilot Chat** — 自然言語での対話的支援
- ✅ **Inline Edits** — 既存コードの直接修正
- ✅ **Agent Mode** — 複数ファイルまたがる自動実装

---

## 🎓 学習の進め方

### 推奨ルート: シンプルから複雑へ

```
① Inline Completions
   ↙
② Chat / Inline Edits（並列可）
   ↙
③ Agent Mode
```

**理由**:
1. **Completions** は最も基本的で日常的
2. **Chat & Edits** はコーダーとしてのペースで学べる
3. **Agent** は複数機能の応用なので最後

---

## 💡 各機能の使い分け

| 機能 | 最適なシーン | 所要時間 | 複雑さ |
|------|-----------|--------|------|
| **Completions** |「次の行を書きたい」「パターン補完」 | 数秒 | ⭐ |
| **Chat** | 「書き方を教えて」「なぜ？」 | 1-5 分 | ⭐⭐ |
| **Inline Edits** | 「このコード改善して」（範囲限定） | 30-60 秒 | ⭐⭐ |
| **Agent** | 「この要件で複数ファイル実装」 | 5-15 分 | ⭐⭐⭐ |

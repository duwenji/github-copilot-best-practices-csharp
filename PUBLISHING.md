# PUBLISHING

## Pre-publish flow
1. `build.json` / `metadata.yaml` / `github-copilot-best-practices-csharp.kdp.yaml` を確認する
2. ebook ビルドを実行して `ebook-output/` の以下を確認する
   - `github-copilot-best-practices-csharp.epub`
   - `github-copilot-best-practices-csharp.pdf`
   - `github-copilot-best-practices-csharp-kdp-registration.md`
3. `VALIDATION_CHECKLIST.md` を使って導線・出力物・KDP 登録情報を確認する
4. リリースノートを整えて公開する
